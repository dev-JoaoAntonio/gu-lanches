# Lanches do Gú

Site institucional do **Lanches do Gú** — hambúrgueres artesanais.

Construído com Vue 3 + Quasar Framework + Vite.

## Rodando localmente

```bash
npm install
npm run dev
```

Abre em `http://localhost:5173`.

## Build de produção

```bash
npm run build
npm run preview
```

## Stack

- Vue 3 (Composition API)
- Quasar Framework v2
- Vite 6
- SCSS
- IntersectionObserver para reveals
- Scroll-driven animations nativas com fallback
